# Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow using real-world data. It includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation in Power BI, report generation, and presentation creation using Gamma.

The goal of this project is to transform raw data into meaningful insights and support data-driven decision-making.

## Dataset
The dataset contains structured business data such as sales, customers, products, revenue, and regional performance.

**Data Source:** CSV / Excel / Database Tables

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- PostgreSQL / MySQL / SQL Server
- Power BI
- Gamma

## Project Steps

### 1. Data Loading
- Imported dataset into Python
- Read CSV / Excel files
- Connected database sources

### 2. Data Cleaning
- Removed duplicates
- Handled missing values
- Fixed data types
- Standardized columns

### 3. Exploratory Data Analysis (EDA)
- Summary statistics
- Trend analysis
- Correlation checks
- Outlier detection
- Business insights

### 4. SQL Analysis
Used SQL queries for:
- Top-selling products
- Monthly sales trends
- Region-wise revenue
- Customer analysis

### 5. Power BI Dashboard
Created an interactive dashboard with:
- KPI Cards
- Sales Trends
- Category Analysis
- Regional Performance
- Filters and Slicers

### 6. Reporting & Presentation
- Final business report created
- PPT prepared using Gamma

## Results
- Identified top-performing products and regions
- Improved understanding of customer behavior
- Highlighted low-performing areas
- Generated actionable business insights
